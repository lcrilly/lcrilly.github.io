![logo](assets/img/lps_logo_2026_light.png#gh-light-mode-only)
![logo](assets/img/lps_logo_2026_dark.png#gh-dark-mode-only)

This is the source of the Litlington Pre-school website hosted at [www.litlingtonpreschool.net](https://www.litlingtonpreschool.net/).

### The stack

* All content in [Markdown](https://www.markdownguide.org/)
* [Jekyll](https://jekyllrb.com/) static site generator
* Hosted with [GitHub Pages](https://pages.github.com/)

### Local development

Start a development container with Docker or Apple Container with [container-compose](https://github.com/Mcrich23/Container-Compose).

```shell
docker compose up --build -d # Docker
container-compose up --build -d # Apple Container
```
